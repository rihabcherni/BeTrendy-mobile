### 3ï¸âƒ£ Setup Backend (Django + MongoDB)

```bash

# Start development server

```

Access the API via: **[http://localhost:8000](http://localhost:8000)**  
Access admin panel via: **[http://localhost:8000/admin](http://localhost:8000/admin)**
